Version : 1.1.1

(주)오픈소스컨설팅에서 자체 웹앱 개발시 사용되는 기본적인 디자인 자료입니다. 오픈 소스로 공개되었으며 자유롭게 사용하셔도 됩니다. 보다 자세한 내용은 아래 본문을 확인해주세요.

  • NPM 패키지로 배포되어 있으며 손쉽게 설치가 가능하고 헤당 링크로 이동해 설치 방법을 확인할 수 있습니다.
  • 스포카 한 산스(SpoqaHanSans) 폰트를 기본으로 사용합니다.
  • OSC-Style의 디렉토리 구조는 이렇게 되어 있습니다.
    • osc-style
      • build
      • config
      • example
      • node_modules
      • style_variables
        • _osc-color.scss
        • _osc-font.scss
      • etc...
  • osc-style안의 style_variables폴더의 파일들을 import해서 사용하면 됩니다.

Usage

Example for SCSS :
 @import 'path/osc-style/style_variables/osc-color'; @import 'path/osc-style/style_variables/osc-font'; * {   @extend .osc-font; } h1 {   color: $primary; }
